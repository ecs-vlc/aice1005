Queues

We are going to implement a queue using a circular array (we could use a linked list, but won't). We will use the conventional names enqueue(const T& x) and dequeue() to add and remove items from the queue. We could implement this using an array

This would however be hugely wasteful of memory. Instead we are going to use a resizable circular array.


  1. I provide you with a header file Queue.h, an implementation file Queue.cc a test file test_queue.cc and a second header file to do unit testing and a makefile. Download these to a directory.
  2. You should be able to compile these using the command make to get an executably file test_queue.
  3. If you run test_queue it should tell you that you have failed all the tests. test_queue uses doctest to do unit testing.
  4. Your job is to fill in the implementation in Queue.cc so thatthe code passes all the tests. The default capacity should be 8. If there are more than 8 items in the queue you should double the capacity.
